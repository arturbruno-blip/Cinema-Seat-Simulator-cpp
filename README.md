# 🎬 Cinema Seat Simulator (C++)

A simple console-based cinema seat reservation system written in C++.

## 📌 Features
- Display seat layout (25x30)
- Select seats by row and column
- Prevent selecting occupied seats
- Input validation

## 🛠️ Technologies
- C++
- Standard Library (iostream)

## ▶️ How to Run
1. Compile:
   g++ main.cpp -o cinema
2. Run:
   ./cinema

## 📷 Example
Shows a grid where:
- 0 = available
- 1 = occupied
